# Temperature Converter

A simple Python program to convert temperatures between Celsius, Fahrenheit, and Kelvin.

## Features
- Celsius to Fahrenheit
- Fahrenheit to Celsius
- Celsius to Kelvin
- Kelvin to Celsius

## How to Use
1. Run: `python temp_converter.py`
2. Choose conversion type (1-4)
3. Enter temperature value
4. Get result

## Example
```
Enter 100°C → Get 212°F
Enter 0°C → Get 273.15K
```
